# Guessy
Game for guessing a randmon number within 12 of less tries

## Tecnologies
- Java 17
- Java Swing

## Guide of ussage 
- download Java JDK 17 or more
- clone the repo
  
``` bash
$ git clone https://github.com/santiagogr05/Guessy.git
```
- Navigate into Guessy/out/artifacts folder
- Execute the .jar file
